# LightAI Go

LightAI Go 是一个参考 GPUStack 架构与能力、面向中小型客户的轻量 AI 资源与模型服务管理平台。

## 第一阶段目标

第一阶段优先实现：

1. Server / Agent 基础架构；
2. GPU 服务器接入；
3. GPU 资源监控；
4. 模型定义管理；
5. 运行环境管理；
6. 模型实例创建；
7. 模型实例启动、停止、重启；
8. 实例状态、日志、错误信息可见；
9. 模型服务 endpoint 可访问。

## 第一阶段暂不实现

第一阶段暂不实现：

- Kubernetes；
- Ray；
- 多集群；
- 复杂调度；
- 模型市场；
- API Key；
- Token 统计；
- 额度管理；
- 成本核算；
- 复杂统一网关。

## 参考原则

GPUStack 仅作为架构和能力参考，不复制代码，不逐行翻译。

LightAI Go 优先面向数台 GPU 服务器的中小客户现场，强调部署简单、管理清楚、运行稳定、现场可排障。
