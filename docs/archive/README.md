# Documentation Archive

> Status: CURRENT
> Last reviewed: 2026-06-18
> Scope: Archive directory rules
> Read order: See `docs/CURRENT.md`

This directory contains historical documents, superseded designs, and old plans.

## Rules

1. Archive documents are for historical traceability only.
2. Archive documents must not be used as current implementation guidance.
3. If an archive document conflicts with `docs/CURRENT.md`, `docs/CURRENT.md` wins.
4. Restoring an archived design requires a new review.

## Subdirectories

| Directory | Purpose |
| --- | --- |
| `superseded/` | Designs replaced by current design documents |
| `historical/` | Historical RC/review artifacts |
| `old-plans/` | Closed or superseded implementation plans |
| `old-reports/` | Reserved for old reports not tied to report evidence directories |

## Naming Rule

Use lowercase kebab-case for new archive files when practical. Existing historical evidence may keep its original name when renaming would reduce traceability.
